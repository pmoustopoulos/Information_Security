# Information Security #
## Implementing a Simple Cryptotool ##

This application is used to encrypt and decrypt **text files**. In order to do that specific steps have to be followed.
The steps that have to be followed are:<br>
1. The user presses the button with the three dots "**...**".
2. Selects the text file that they want to encrypt.
3. The text of the selected file appears inside the Plain Text in the left **TextArea**.
4. Assuming the text file is not encrypted, the user will try to encrypt it.
5. The user presses the button "**Encrypt**" to encrypt the file. The encrypted text appears in the right **TextArea**.
    * If a file was **not selected** and the user presses any of the two buttons "Encrypt" or "Decrypt" then a pop-up 
    window appears to inform them about their incorrect action.
6. In order to decrypt the text file the user follows the **same steps** but instead of pressing the "**Encrypt**" 
button, they should press the "**Decrypt**" button.
