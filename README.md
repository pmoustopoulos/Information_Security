# Information Security #
## Implementing a Simple Cryptotool ##

This application is used to encrypt and decrypt **files**. In order to do that specific steps have to be followed.
The steps that have to be followed are:<br>
1. The user presses the button with the three dots "**...**".
2. The user selects the file that they want to encrypt.
3. As soon as the user selects a file, inside the textfield the path of the file will appear.
4. Then the user presses the button "**Encrypt**" and a pop-up window will be displayed to inform them
   that their action was successfully accomplished.
    * If a file was **not selected** and the user presses any of the two buttons "Encrypt" or "Decrypt" then a pop-up 
    window appears to inform them about their incorrect action.
6. In order to decrypt the file, the user follows the **same steps** but instead of pressing the "**Encrypt**" 
button, they should press the "**Decrypt**" button.
